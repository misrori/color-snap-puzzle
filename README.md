# Tetra Puzzle

A classic block-dropping puzzle game. Clear lines, score points, and master the challenge!

## Features
- Smooth block-dropping mechanics
- Multi-level gameplay
- Touch and keyboard controls
- High-quality neon aesthetics
- Performance optimized for mobile and web

## Deployment
- Web: Hosted on GitHub Pages
- Android: Play Store ready via Capacitor

## Local Development
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`
